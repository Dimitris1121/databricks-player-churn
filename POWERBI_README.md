# Online Player Analytics Dashboard — Power BI

Interactive dashboard built in Power BI for online gaming player analytics.

## What it shows
- **Total Players, Churn Rate, Avg Deposits** — KPI cards with DAX measures
- **Churn Rate by Segment** — engagement segmentation (High/Medium/Low)
- **Revenue by Game Category** — conditional formatting showing top performers
- **Players by Device** — Mobile/Desktop/Tablet breakdown
- **Interactive slicers** — filter by Month and Country

## Key findings
- Sports Betting and Slots drive the highest revenue
- 30% overall churn rate across all segments
- 56% of players use Mobile devices

## Stack
Power BI Desktop · DAX · Power Query · Synthetic player dataset
